项目收获：  
（1）对Vue这种MVVM组件化开发思想，有了深刻的认识。可以熟练地进行组件注册，组件通信传值，组件抽象复用，路由的配置，以及对vue生命周期函数的认识和理解。  
（2）对css3的布局，样式书写，动画，有了很大的提高。  
（3）对于移动端开发有了更深的认识。
